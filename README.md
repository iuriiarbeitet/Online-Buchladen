# Online Book Store

Acesta este un proiect pentru un magazin online de cărți construit cu Spring Boot, Java și Thymeleaf.

## Funcționalități
- Gestionarea utilizatorilor
- Catalog de cărți
- Coș de cumpărături și finalizare comandă

## Tehnologii folosite
- Java
- Spring Boot
- Spring Security
- Thymeleaf
- Hibernate / JPA
